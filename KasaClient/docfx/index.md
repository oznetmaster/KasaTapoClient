# KasaTapoClient

KasaTapoClient is a .NET client library for TP-Link Kasa and Tapo devices, supporting discovery, inspection, refresh, and control of plugs, bulbs, light strips, strips, hubs, and selected child devices.

## Supported frameworks

- `.NET 10`
- `.NET Framework 4.7.2`

## Packages and projects

- `KasaClient` - reusable client library
- `KasaClient.Console` - console client for discovery and host/light/child operations
- `KasaClient.Tests` - deterministic and optional live integration coverage
